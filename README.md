# EDC-experiments

This repository contains all the code to generate the plots and tables used in the `EDC: Equation Discovery for Classification` paper.

## Installation of EDC

This repository is part of the EDC project.
To install the EDC-core package run the following command:

```bash
pip install -e <folder-path>
```

Where `<folder-path>` is the path to the EDC-core folder.


## Optimisers

This folder contains the code to run the optimiser experiments described in the paper.

## Artifical

This folder contains the code to run the artificial experiments described in the paper.

## Real world

This folder contains the code to run the real world experiments described in the paper.
